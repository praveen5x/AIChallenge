# 🎮 Emoji Dodge – AI Pilot

A fast-paced arcade-style Android game built using **Jetpack Compose** where you control a robot 🤖 and dodge falling emojis.

---

## 🚀 Features

- 🎯 Smooth gesture-based player movement
- ⚡ Real-time game loop using `withFrameNanos`
- 🧠 Light AI assist to help avoid obstacles
- ❤️ Lives system with temporary invincibility
- 📈 Dynamic difficulty scaling
- 🎨 Beautiful Compose UI with gradients & animations
- 💥 Game Over screen with replay option

---

## 🕹️ Gameplay

- Drag your finger to move the robot 🤖
- Avoid falling emojis 😈👻💀🔥⚡
- Each dodge increases your score
- You have **3 lives**
- After getting hit, you become temporarily invincible
- Survive as long as possible!

---

## 🏗️ Tech Stack

- **Kotlin**
- **Jetpack Compose**
- **Canvas API** (custom rendering)
- **Material 3**
- **Coroutines (`LaunchedEffect`)**
- **Custom Game Loop (120Hz physics)**
